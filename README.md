🛠️ Sistema de Interventoría - Security Shops

Sistema web desarrollado en PHP para la gestión integral de proyectos de interventoría, permitiendo el control, seguimiento y administración en tiempo real mediante asignación de responsables y trazabilidad completa de cada proyecto.

🚀 Demo en vivo
👉 https://interventoria.infinityfree.me/


📌 Características

🔐 Sistema de autenticación de usuarios
📊 Dashboard con estadísticas de proyectos
📁 Creación de proyectos por parte del jefe de implementaciones
👤 Asignación de proyectos a interventores
💬 Sistema de comentarios (hilo por proyecto) para seguimiento continuo
🧵 Trazabilidad completa desde la creación hasta el cierre del proyecto
🏷️ Clasificación por categorías y prioridad
📈 Visualización de estado (abiertos / cerrados)

🧠 Tecnologías utilizadas

PHP (Backend)
MySQL (Base de datos)
HTML, CSS, JavaScript
Bootstrap
jQuery

⚙️ Instalación local

Clonar el repositorio:
git clone https://github.com/Andrescutellez/sistema-tickets.git

Importar la base de datos .sql en phpMyAdmin

Configurar conexión en:
config/conexion.php

Ejecutar en XAMPP:
http://localhost/Interventoria
